<h1 align="center">🏓 Pong Tournament Web App</h1>

<p align="center">
  A modern <strong>Single Page Application</strong> for playing Pong — developed as part of a full-stack web project.<br/>
  Special thanks to <strong>@emilmirzazada</strong> for project guidance and <strong>@CodeNikkiya</strong> for the custom Pong game.
</p>

<hr/>

<h2>🚀 Features</h2>

<ul>
  <li>🧑‍💻 Register players and manage profiles</li>
  <li>🎮 Play <strong>Pong vs AI</strong> or <strong>1v1 on the same keyboard</strong></li>
  <li>🏆 Host a <strong>4-player tournament</strong></li>
  <li>📊 View match results and <strong>player statistics</strong> on dashboards</li>
  <li>🔐 All data is saved in a <strong>PostgreSQL</strong> database</li>
</ul>

<h2>🛠️ Tech Stack</h2>

<ul>
  <li><strong>Frontend:</strong> Vanilla JavaScript SPA</li>
  <li><strong>Game Engine:</strong> Custom Pong by @CodeNikkiya</li>
  <li><strong>Backend:</strong> Python + Django</li>
  <li><strong>Database:</strong> PostgreSQL</li>
  <li><strong>Containerization:</strong> Docker & Docker Compose</li>
</ul>

<h2>🧪 Getting Started</h2>

<p>To run the project locally:</p>

<strong>docker-compose up --build</strong>
<p>Then open your browser and go to:</p>
<strong>http://localhost</strong>
